# 项目说明

WordCount -> 词频统计

InvertedIndex -> 倒排索引

Serializetion -> 序列化

Partition -> 自定义分区

sort -> 自定义排序,
        其中AllSort只有一个分区，对该分区排序；
        PartitionSort既有自定义分区，又有分区内的排序

Count -> 带有Combiner的WordCount

outputFormatDemo -> 自定义Reducer结束之后的输出格式和位置

lab1 -> 实验要求在文件夹里面有

TableJoin -> 使用MR处理表的连接问题
